Welcome!

Welcome to the Downriver Fantasy Football companion site! This is meant to be viewed as just that, a companion to the ESPN main site. When you close this popup window, the following features are available on the website:

  • Current Season - Will show standings, points for/against, Optimal Record (what everyone's record would be if they set the best lineups), and win streak. Similar features can be found on the ESPN site.
  • Records & History - This will show all-time statistics, as well as statistics from each season. It shows columns such as all-time win percentage, Playoff records, and championships.
  • All Time Records - Since the inception of this league in 2011, it shows all-time records in various categories.
  • Head-to-Head - You can find out any owner's record vs anyone all time with this tool!
  • Championships - Exactly what it says.

This site is viewable on all platforms (mobile, tablets, PCs, Macs etc). It is in no way affiliated with ESPN.
